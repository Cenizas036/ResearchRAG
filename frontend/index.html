Search

Hybrid Search

Combines dense vector embeddings (Qwen 2.5 72B / sentence-transformers) with BM25 sparse retrieval for best-in-class recall. Query across thousands of research papers in milliseconds.

Retrieval

Smart Retrieval

AI-powered re-ranking with cross-encoder models ensures the most relevant chunks surface first. Supports ChromaDB, FAISS, and Pinecone vector stores out of the box.

retriever = HybridRetriever(
  model: Qwen2.5-72B-Instruct,
  sparse: BM25(k1=1.5),
  reranker: CrossEncoder("ms-marco")
)
Citations

Citation Tracking

Every answer is grounded in source documents. Auto-generates citations with page numbers, DOI links, and confidence scores so you can verify every claim.

Reasoning

Multi-hop Reasoning

Chain-of-thought reasoning with iterative retrieval. The system breaks complex questions into sub-queries, retrieves context for each, and synthesizes a comprehensive answer.

Support

Frequently Asked Questions

Everything you need to know about the product.

ResearchRAG supports PDF documents, ArXiv papers (via ID or URL), web pages, and DOI lookups. The ingestion pipeline extracts text, splits it into semantic chunks, and indexes them for retrieval.
ResearchRAG supports ChromaDB for local development, FAISS for high-performance in-memory search, and Pinecone for scalable cloud-hosted vector storage. Switch between them with a single config change.
Every generated answer is cross-referenced against the retrieved source chunks. Claims that cannot be traced back to a specific document are flagged with a confidence warning, ensuring research integrity.
ResearchRAG is powered by Qwen 2.5 72B Instruct, a state-of-the-art open-source large language model hosted for free on Hugging Face. No paid API keys required — it's completely free to use.
